간편 입력

연말정산 입력

총급여·카드사용·연금저축만 입력해도 환급/추가납부 예상액과 절세 포인트를 안내합니다.

결과/피드백

환급·절세 인사이트

계산을 실행해 주세요.

총급여, 원천징수, 카드 사용액을 입력하면 예상 환급/추가 납부액을 표시합니다.

법인세

손익·조정 입력

접대비 한도, 감가상각, 최저한세, 세액공제를 한 번에 확인합니다.

결과/피드백

법인세 인사이트

계산을 실행해 주세요.

접대비 한도, 최저한세, 세액공제를 반영한 예상 납부/환급액을 보여줍니다.

금융소득 종합과세

금융소득 · 기타소득 입력

20,000,000원 기준 비교과세와 배당 Gross-up, 외국납부세액공제까지 한 번에 확인합니다.

금융소득 항목

이자/배당별 금액과 원천세율, 국내·해외 여부를 입력하세요.

유형 금액 원천세율 원천 Gross-up 기납부/외국세액

입력 팁

  • 해외 배당/이자는 원천을 “해외”로 선택하면 종합과세로 강제 전환됩니다.
  • Gross-up 대상 배당만 체크하면 배당가산·배당세액공제를 자동 계산합니다.
  • 금융소득이 20,000,000원 이하라도 해외소득이 있으면 비교과세를 수행합니다.

배포 가이드

GitHub → Cloudflare Pages

정적 사이트라 빌드 단계 없이 바로 배포할 수 있습니다.

  1. 이 폴더를 GitHub에 올립니다. (루트에 index.html, styles.css, app.js 유지)
  2. Cloudflare Pages에서 새 프로젝트를 만들고 리포지토리를 연결합니다.
  3. Build command: 없음, Output directory: /
  4. 배포 후 각 탭 계산기를 열어 결과가 나오는지 확인합니다.
