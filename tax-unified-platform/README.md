# 통합 세금 플랫폼

연말정산 · 법인세 · 금융소득 종합과세 계산기를 한 화면에 묶은 정적 웹 앱입니다. 핵심 숫자만 입력하면 즉시 예상 납부/환급액과 절세 포인트를 안내하도록 UI를 단순화했습니다.

## 특징
- **연말정산**: 총급여·원천징수·카드사용·연금저축 입력으로 환급/추가납부 예상, 카드 공제 여지와 연금 공제 가능액을 바로 표시.
- **법인세**: 접대비 한도, 감가상각 불산입, R&D/투자 세액공제, 최저한세를 한 번에 확인. 샘플 데이터 버튼 제공.
- **금융소득 종합과세**: 2천만원 한도 비교과세, 배당 Gross-up/배당세액공제, 해외소득·외국납부세액공제 계산. 금융소득 행을 동적으로 추가.

## 파일 구성
- `index.html` : 탭 기반 UI와 배포용 엔트리
- `styles.css` : 다크 톤 그라디언트·카드 레이아웃 스타일
- `app.js` : 세 가지 계산 로직, 샘플 데이터, 결과 렌더링

## 로컬 확인
정적 사이트이므로 간단히 미리보기할 수 있습니다.
```bash
cd tax-unified-platform
python -m http.server 8787
# 브라우저에서 http://localhost:8787
```

## 배포 (GitHub → Cloudflare Pages)
1. 이 폴더를 GitHub 리포지토리 루트에 올립니다.
2. Cloudflare Pages에서 새 프로젝트 생성 후 해당 리포지토리를 연결합니다.
3. Build command: 없음 / Output directory: `/`
4. 배포 후 각 탭(연말정산, 법인세, 금융소득)을 열어 샘플 데이터 → 계산 순서로 표시되는지 확인합니다.

## 입력 가이드
- **연말정산**: 비과세 제외 총급여, 원천징수세액, 카드/체크/전통시장/문화비, 연금저축·IRP, 주택자금 관련 금액, 부양가족 수만 있으면 계산 가능.
- **법인세**: 당기순이익, 매출(일반/특수관계), 접대비 세부, 감가상각, 가지급금·이자율, 이월결손금, R&D/투자 공제, 기납부세액을 넣으면 핵심 흐름을 따라 계산.
- **금융소득**: 이자/배당별 금액과 원천세율, 국내/해외 여부, Gross-up 대상 체크, 기타 종합소득과 소득공제 합계, 기납부세액을 입력.

배포나 스타일을 수정해도 계산 로직은 `app.js`에만 존재하므로 유지보수가 간단합니다.
